# Location Sharing App 1

A mobile app for real-time location sharing with friends, built using **Kotlin** and **Firebase**. Users can sign in, view friends, share locations, and update profiles.

---

## Features

- Firebase Authentication (Sign In / Sign Up)
- Real-time location tracking
- Profile management (update name, email, location)
- Friend list with real-time updates
- Map view to see friends' locations
- Navigation drawer with quick access to profile, map, and logout

## Installation

1. Clone the repository:
```bash
git clone https://github.com/koraish3438/Location-Sharing-App-1.git
```
2. Open in **Android Studio**
3. Sync Gradle and build the project
4. Run on an Android device or emulator

---

## Dependencies

- Firebase Authentication
- Firebase Firestore
- Google Maps SDK
- AndroidX Libraries
- Material Components

---

## Usage

1. Open the app and sign in.
2. View your friend list and their current locations.
3. Tap on the map button to see friends on the map.
4. Update your profile details via the profile screen.
5. Logout when done.
